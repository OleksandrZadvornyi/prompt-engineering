As an agency user, I want to create a submission with my provided data, so that I can have it validated and published.
As a broker user, I want to submit data to the system, so that it can be processed and published if valid.
As a developer, I want to access broker application data, so that I can work with the submitted information.
As a developer, I want to prevent double publishing of submissions, so that I avoid errors due to multiple submissions.
As a FABS user, I want to submit a FABS file, so that my specific data can be processed and validated.

As a submission object, I want to validate the submission data, so that I can determine if it is valid or contains errors.
As a submission object, I want to update my last modified timestamp when published, so that the latest modification time is recorded.
